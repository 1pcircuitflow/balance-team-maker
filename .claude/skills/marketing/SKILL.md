# /marketing - 마케팅 및 ASO 어시스턴트

Balance Team Maker (Belo)를 위한 앱 스토어 최적화(ASO) 콘텐츠, 마케팅 문구, 업데이트 노트 및 키워드를 생성합니다.

## 사용법
```
/marketing <서브커맨드> [옵션]
```

서브커맨드: `aso`, `update-note`, `description`, `keywords`

## 인수 ($ARGUMENTS)
$ARGUMENTS

## 앱 컨텍스트
- **앱 이름:** Belo (Balance Team Maker)
- **카테고리:** 스포츠 / 유틸리티
- **타겟 사용자:** 아마추어 스포츠 팀 주최자 (축구, 풋살, 농구)
- **핵심 가치:** 실력 티어 및 포지션 선호도를 기반으로 한 공정한 팀 밸런싱
- **시장:** 한국 (주요), 브라질, 일본, 미국, 스페인/라틴아메리카
- **언어:** ko, en, es, ja, pt

처리하기 전에 현재 앱 컨텍스트를 위해 다음 파일들을 읽습니다:
- `package.json` - 현재 버전 번호
- `translations.ts` - 모든 언어의 기능 이름 및 설명
- 최근 git 로그 (`git log --oneline -20`) - 최근 변경 사항

## 서브커맨드 지침

---

### `aso` - 앱 스토어 최적화
최대 노출을 위해 Google Play 스토어 등록정보를 최적화합니다.

5개 언어 모두에 대해 생성합니다:
1. **앱 제목** (최대 30자): 주요 키워드 포함
2. **간단한 설명** (최대 80자): 핵심 혜택으로 사용자 유도
3. **그래픽 텍스트:** 홍보용 그래픽을 위한 태그라인

출력 형식:
```
## ASO 최적화

### 한국어 (ko)
- 제목: [제목]
- 간단한 설명: [설명]

### 영어 (en)
...
```

---

### `update-note` - 릴리스 업데이트 노트
최근 git 커밋을 기반으로 업데이트 노트를 생성합니다.

1. `git log --oneline -20`을 실행하여 최근 변경 사항을 확인합니다.
2. 변경 사항을 분류합니다: 새로운 기능, 개선 사항, 버그 수정
3. 개발자 전문 용어가 아닌 사용자 친화적인 업데이트 노트를 작성합니다.
4. 5개 언어 모두로 생성합니다.
5. 각 언어별로 500자 이내로 유지합니다 (Play Store 제한).

출력 형식:
```
## 업데이트 노트 v[버전]

### 한국어 (ko)
[노트]

### 영어 (en)
[노트]
...
```

---

### `description` - 전체 스토어 설명
스토어 등록정보를 위한 매력적인 전체 앱 설명을 작성합니다.

각 언어별 구조:
1. **오프닝 문구** (2-3문장): 문제 제기 + 해결책
2. **주요 기능** (글머리 기호): 5-7가지 주요 기능
3. **사용 방법** (3단계): 간단한 사용자 흐름
4. **지원 스포츠:** 축구, 풋살, 농구
5. **클로징 CTA:** 다운로드 권장

가이드라인:
- 관련 키워드를 자연스럽게 포함합니다.
- 공정성과 밸런싱 알고리즘을 강조합니다.
- 다국어 지원을 언급합니다.
- 전문적이면서도 친근한 어조를 유지합니다.
- 언어당 최대 4000자를 넘지 않습니다.

---

### `keywords` - 키워드 조사 및 제안
ASO 키워드를 분석하고 제안합니다.

1. 현재 앱 콘텐츠(`translations.ts`, `metadata.json`)에서 키워드를 추출합니다.
2. 키워드 카테고리를 제안합니다:
   - 기본: 직접적인 기능 키워드 (팀 메이커, 팀 밸런스 등)
   - 보조: 스포츠별 키워드 (축구 팀, 풋살 라인업 등)
   - 롱테일: 사용 사례 키워드 (공정한 팀 나누기 등)
3. 각 언어별로 관련성 순으로 상위 15개 키워드를 제안합니다.
4. Play 콘솔에 직접 붙여넣을 수 있도록 쉼표로 구분된 형식으로 제공합니다.

---

## 일반 가이드라인
- **어조:** 전문적이면서도 친근하게, 공정성과 사용 편의성을 강조합니다.
- **키워드:** 자연스럽게 통합하며, 키워드 채우기(stuffing)를 지양합니다.
- **현지화:** 직역이 아닌 원어민 수준의 자연스러운 텍스트를 작성합니다.
- **글자 수 제한:** Play 스토어 제한 사항을 엄격히 준수합니다.
- **이모지:** 설명문에서 절제하여 사용합니다 (섹션당 최대 1-2개).

## 중요 참고 사항
- 스토어에서는 "Belo"라고 불리지만 "Balance Team Maker"가 설명적인 이름입니다.
- 주요 차별점: 알고리즘 기반의 공정한 팀 밸런싱 (랜덤 아님)
- 프로 기능: 광고 제거, 무제한 포지션 밸런싱, 클라우드 동기화
- 앱은 팀 제약 조건(함께 편성 / 분리 편성)을 지원합니다.
